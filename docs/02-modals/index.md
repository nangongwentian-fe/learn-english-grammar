# 二、情态动词 Modal Verbs

::: tip 什么是情态动词？
中文里我们说"我**能**游泳"、"你**必须**写作业"、"他**可能**迟到"——这些"能、必须、可能"就是情态的概念。英语里有一组特殊的动词专门表达这些意思，叫做 **情态动词（Modal Verbs）**。它们不单独使用，而是**放在主要动词前面**，给动词加上"能力、许可、义务、可能性"等含义。
:::

## 情态动词的共同特点

在学习每个情态动词之前，先记住它们的**共同规则**：

| 规则 | 说明 | 例子 |
|------|------|------|
| 后面接**动词原形** | 不加 to、不加 ing、不加 ed | She **can swim**. (不是 ~~can to swim~~) |
| **没有**第三人称单数变化 | 不加 s/es | He **can** speak English. (不是 ~~He cans~~) |
| 否定：直接加 **not** | 不需要助动词 do/does | You **must not** lie. (不是 ~~don't must~~) |
| 疑问：情态动词**提前** | 不需要助动词 do/does | **Can** you help me? (不是 ~~Do you can~~) |
| **没有**完整时态变化 | 没有 -ing 形式，没有过去分词 | 不存在 ~~canning~~ 或 ~~musted~~ |

## 雅思考试中的情态动词

情态动词是雅思评分标准中 **Grammatical Range and Accuracy（语法多样性与准确性）** 的重要考察点。

- **写作 Task 2**（议论文）：用 should / must / may 表达观点和建议，用 could / might 表达不确定性，展示语法灵活性
- **写作 Task 1**（图表题）：用 may / could 描述数据趋势中的可能原因
- **口语 Part 2/3**（讨论话题）：用 would / used to 谈过去习惯，用 should / could 给建议，用 might 表推测
- **阅读/听力**：理解文中情态动词的语义差异是正确作答的关键

::: warning 考生常见问题
很多中国考生在写作中只会用 can 和 should，其他情态动词几乎不用。这会导致语法评分偏低。学完这个模块后，你应该能在写作和口语中**灵活使用至少 6-8 个情态动词**。
:::
