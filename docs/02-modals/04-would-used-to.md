# 过去习惯 Past Habits

::: tip 核心概念
中文说"我小时候**经常**去奶奶家"、"她以前**常常**弹钢琴"——这些描述"过去的习惯"的表达，在英语中用 **used to** 和 **would** 来表达。而且，中国学生最容易搞混的是 **used to / be used to / get used to** 这三个长得很像但意思完全不同的短语！
:::

## 1. used to — 过去常常（但现在不了）

### 什么时候用？

**used to** 表达**过去经常做但现在不再做的事情**，或者**过去的状态但现在已经改变了**：

1. **过去的习惯动作** — 以前经常做某事（现在不做了）
2. **过去的状态** — 以前是某种状态（现在不是了）

> 核心要点：**"以前...但现在不了"，用 used to。一定包含"变化"的含义。**

### 怎么构成？

#### 肯定句

| 构成 | 例句 | 中文 |
|------|------|------|
| 主语 + **used to** + 动词原形 | I **used to play** football every weekend. | 我以前每个周末**都踢**足球。（现在不踢了） |
| | She **used to live** in Shanghai. | 她以前**住在**上海。（现在不住了） |
| | There **used to be** a park here. | 这里以前**有**一个公园。（现在没有了） |

::: warning 中国人高频错误
**used to 后面接动词原形，不加 -ing！**

<div class="example-wrong">❌ I used to <strong>playing</strong> football.</div>
<div class="example-right">✅ I used to <strong>play</strong> football.</div>

**是 used to，不是 use to（肯定句中）！**

<div class="example-wrong">❌ I <strong>use to</strong> live here.</div>
<div class="example-right">✅ I <strong>used to</strong> live here.</div>
:::

#### 否定句

否定句有两种写法：

| 构成 | 例句 | 中文 |
|------|------|------|
| 主语 + **didn't use to** + 动词原形 | I **didn't use to** like vegetables. | 我以前**不喜欢**吃蔬菜。（现在喜欢了） |
| 主语 + **never used to** + 动词原形 | She **never used to** exercise. | 她以前**从不**锻炼。（现在锻炼了） |

::: info 注意！
否定句用 didn't 时，used 变回原形 **use**（因为 didn't 已经表示过去时了）：
- ✅ I **didn't use to** like coffee.
- 也有人写 I **didn't used to**... — 两种都有人用，考试中用 **didn't use to** 更安全。
:::

#### 疑问句

| 构成 | 例句 | 中文 |
|------|------|------|
| **Did** + 主语 + **use to** + 动词原形？ | **Did** you **use to** play tennis? | 你以前**常打**网球吗？ |
| | **Did** she **use to** live here? | 她以前**住**这里吗？ |

### used to 的用法举例

| 过去（used to） | 现在 |
|----------------|------|
| I **used to smoke**. | Now I don't smoke.（我戒烟了。） |
| She **used to have** long hair. | Now she has short hair.（她现在短发了。） |
| We **used to walk** to school. | Now we take the bus.（现在坐公交了。） |
| This area **used to be** farmland. | Now it's a shopping mall.（现在是商场了。） |

### 练习题

  question="I ___ (used to / eat) a lot of fast food, but now I eat healthy."
  answer="used to eat"
  hint="过去的习惯，现在不了：used to + 动词原形"
/>

<Quiz
  question="___ you ___ (use to / play) any sports when you were young?"
  answer="Did you use to play"
  hint="疑问句：Did + 主语 + use to + 动词原形"
/>

<Quiz
  question="There ___ (used to / be) a cinema here, but it closed down."
  answer="used to be"
  hint="过去的状态：used to + be"
/>

## 2. would — 过去经常做的动作

### 什么时候用？

**would** 也可以表示**过去重复的动作/习惯**，但与 used to 有一个重要区别：

- **would** 只能表示**过去反复的动作**
- **would 不能**表示**过去的状态**

> 简单记忆：**would = 过去反复的"做"，不能用于"是/有/住在"等状态。**

### 怎么构成？

| 构成 | 例句 | 中文 |
|------|------|------|
| 主语 + **would** + 动词原形 | Every summer, we **would visit** my grandparents. | 每年夏天，我们**都会**去看望爷爷奶奶。 |
| | On Sundays, my father **would cook** a big dinner. | 每到周日，爸爸**总会**做一顿丰盛的晚餐。 |
| | As children, we **would play** in the garden for hours. | 小时候，我们**总是**在花园里玩好几个小时。 |

### used to vs would — 核心区别

| | used to | would |
|---|---------|-------|
| 过去的**动作**（习惯） | ✅ I **used to play** tennis. | ✅ I **would play** tennis. |
| 过去的**状态** | ✅ I **used to live** in Beijing. | ❌ ~~I would live in Beijing.~~ |
| 暗示"现在不了" | ✅ 强烈暗示 | ❌ 不一定暗示 |
| 需要时间背景 | ❌ 可以独立使用 | ✅ 通常需要提供时间背景 |

::: danger would 不能用于状态！
以下动词表示**状态**，只能用 **used to**，不能用 would：

**be, have, live, know, like, love, want, believe, belong, own**

<div class="example-wrong">❌ I <strong>would be</strong> very shy as a child.</div>
<div class="example-right">✅ I <strong>used to be</strong> very shy as a child.（我小时候很害羞。）</div>

<div class="example-wrong">❌ She <strong>would have</strong> a red bicycle.</div>
<div class="example-right">✅ She <strong>used to have</strong> a red bicycle.（她以前有一辆红色自行车。）</div>

<div class="example-wrong">❌ We <strong>would live</strong> in a small town.</div>
<div class="example-right">✅ We <strong>used to live</strong> in a small town.（我们以前住在小镇上。）</div>
:::

::: info would 通常需要时间背景
would 表示过去习惯时，句子中通常需要有**时间背景**来提供语境，否则容易和 would 的其他用法（如虚拟语气）混淆。

- ✅ **When I was young**, I **would** spend hours reading.（时间背景：When I was young）
- ✅ **Every summer**, we **would** go camping.（时间背景：Every summer）
- ❌ I **would** go camping.（没有时间背景，意思不明确 — 可能被理解为"我愿意去"）
:::

### 练习题

<Quiz
  question="When I was a child, I ___ (would / spend) every summer at my grandmother's house."
  answer="would spend"
  hint="过去反复的动作 + 有时间背景：would + 动词原形"
/>

<Quiz
  question="She ___ (used to, have) a dog, but it passed away. (状态，不能用 would)"
  answer="used to have"
  hint="have 是状态动词，只能用 used to，不能用 would"
/>

## 3. used to vs be used to vs get used to — 超高频考点！

这三个短语**长得非常像**，但意思**完全不同**！这是雅思考试和各类英语考试中的超高频考点。

### 三者对比

| 短语 | 意思 | 后面接什么 | 时态 |
|------|------|-----------|------|
| **used to** + 动词原形 | 过去常常做（现在不了） | 动词原形 | 只有过去 |
| **be used to** + 名词/doing | 习惯于（已经适应了） | 名词 或 动词-ing | 任何时态 |
| **get used to** + 名词/doing | 逐渐习惯（适应的过程） | 名词 或 动词-ing | 任何时态 |

### used to — 过去常常

| 例句 | 中文 |
|------|------|
| I **used to** get up at 6am.（动词原形） | 我以前**常常**6点起床。（现在不了） |
| He **used to** drink coffee.（动词原形） | 他以前**常**喝咖啡。（现在不喝了） |

### be used to — 习惯于

**be used to** 表示**已经习惯了**某件事，**不觉得奇怪或困难**了。

| 例句 | 中文 |
|------|------|
| I **am used to** getting up early.（动词-ing） | 我**习惯了**早起。（对我来说不难） |
| She **is used to** the cold weather.（名词） | 她**习惯了**寒冷的天气。 |
| He **wasn't used to** living alone.（动词-ing） | 他**不习惯**独居。 |
| **Are** you **used to** the new job?（名词） | 你**习惯**新工作了吗？ |

::: warning 超级重要
**be used to 后面接 doing（动词-ing）或名词，绝对不接动词原形！**

<div class="example-wrong">❌ I am used to <strong>get up</strong> early.</div>
<div class="example-right">✅ I am used to <strong>getting up</strong> early.</div>

<div class="example-wrong">❌ She is used to <strong>drive</strong> on the left.</div>
<div class="example-right">✅ She is used to <strong>driving</strong> on the left.</div>
:::

### get used to — 逐渐习惯

**get used to** 表示**正在适应/逐渐习惯**某件事的过程。

| 例句 | 中文 |
|------|------|
| I'm **getting used to** my new school.（名词） | 我正在**逐渐适应**新学校。 |
| You'll **get used to** the noise.（名词） | 你会**慢慢习惯**噪音的。 |
| She can't **get used to** eating spicy food.（动词-ing） | 她**无法适应**吃辣的食物。 |
| It took me a while to **get used to** driving on the left.（动词-ing） | 我花了一段时间才**习惯**左侧驾驶。 |

::: info be used to vs get used to
- **be used to** = 已经习惯了（**结果/状态**）
- **get used to** = 正在逐渐习惯（**过程/变化**）

| be used to（已经习惯） | get used to（正在适应） |
|----------------------|----------------------|
| I **am used to** the noise. (已经习惯了) | I'm **getting used to** the noise. (正在适应中) |
| She **is used to** living alone. (已经适应了) | She is **getting used to** living alone. (慢慢在适应) |
:::

### 三者超级对比

看下面三个句子，意思完全不同：

| 句子 | 意思 |
|------|------|
| I **used to live** in Beijing. | 我**以前住在**北京。（现在不住了。used to + 动词原形） |
| I **am used to living** in Beijing. | 我**习惯了住在**北京。（已经适应了。be used to + doing） |
| I **am getting used to living** in Beijing. | 我**正在逐渐适应住在**北京。（还在适应过程中。get used to + doing） |

<div class="example-wrong">❌ I used to living in Beijing.（混淆了 used to 和 be used to）</div>
<div class="example-right">✅ I <strong>used to live</strong> in Beijing.（过去常常：used to + 动词原形）</div>
<div class="example-right">✅ I <strong>am used to living</strong> in Beijing.（习惯了：be used to + doing）</div>

<div class="example-wrong">❌ I am used to get up early.（be used to 后面不能接动词原形）</div>
<div class="example-right">✅ I am used to <strong>getting</strong> up early.</div>

<div class="example-wrong">❌ I used to getting up early.（used to 后面不能接 doing）</div>
<div class="example-right">✅ I <strong>used to get</strong> up early.</div>

<div class="ielts-tip">

**雅思口语 Part 2** 描述过去的经历时用 used to / would：

- *"When I was younger, I **used to** spend a lot of time outdoors."*
- *"My family **would** always get together for dinner on Sundays."*
- *"There **used to be** a beautiful garden near my house, but it's now a car park."*

**雅思口语 Part 3** 谈论适应和变化时用 be used to / get used to：

- *"People in big cities **are used to** fast-paced lifestyles."*
- *"It can take time to **get used to** living in a foreign country."*
- *"Young people **are getting used to** working remotely."*

**雅思写作 Task 2** 描述变化趋势：

- *"People **used to** rely on traditional media, but now most people get news online."*
- *"Students need time to **get used to** the university environment."*
- *"Workers **are now used to** using digital tools in their daily tasks."*

</div>

## 本节总结对比表

### used to vs would

| | used to | would |
|---|---------|-------|
| 过去习惯动作 | ✅ | ✅ |
| 过去状态 | ✅ | ❌ |
| 暗示"现在不了" | ✅ 强 | 不一定 |
| 需要时间背景 | 不需要 | 通常需要 |
| 时态 | 只有过去 | 只有过去 |

### used to / be used to / get used to

| | used to | be used to | get used to |
|---|---------|-----------|------------|
| 意思 | 过去常常 | 习惯于 | 逐渐习惯 |
| 后接 | **动词原形** | **doing / 名词** | **doing / 名词** |
| 时态 | 只有过去 | 任何时态 | 任何时态 |
| 重点 | 过去 vs 现在的对比 | 已适应的状态 | 适应的过程 |
| 例句 | I **used to smoke**. | I **am used to smoking**. | I **got used to smoking**. |
| 中文 | 我以前抽烟（现在不了）。 | 我习惯了抽烟（不觉得难受）。 | 我逐渐习惯了抽烟。 |

::: warning 本节核心错误清单
1. ❌ I used to **playing** → ✅ I used to **play**（used to + 动词原形）
2. ❌ I **use to** live here → ✅ I **used to** live here（肯定句用 used to）
3. ❌ I **would be** shy as a child → ✅ I **used to be** shy（状态不能用 would）
4. ❌ I am used to **get up** early → ✅ I am used to **getting up** early（be used to + doing）
5. ❌ I used to **living** here → ✅ I used to **live** here（used to + 动词原形，不是 doing）
6. ❌ 混淆三个短语 → 牢记：used to + 原形 / be used to + doing / get used to + doing
:::
