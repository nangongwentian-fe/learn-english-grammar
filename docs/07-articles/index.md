# 七、冠词与名词 Articles & Nouns

::: tip 为什么这个模块对中国人特别重要？
中文里没有冠词（a/an/the），也没有"可数/不可数"的语法概念。我们说"给我一个建议"、"买几个家具"，在英语里全是错的！研究表明，**冠词错误是中国雅思考生写作中最常见的语法错误类型，没有之一。** 如果你能掌握冠词和名词的规则，你的写作分数至少能提高 0.5 分。
:::

## 雅思中的冠词与名词

<div class="ielts-tip">

**冠词是雅思写作评分标准 "Grammatical Range and Accuracy" 中的重点考察项。** 考官明确指出：

- **Band 5**：频繁出现冠词和名词复数错误
- **Band 6**：冠词错误较多但不影响理解
- **Band 7**：冠词使用基本准确，偶有错误
- **Band 8+**：冠词使用几乎没有错误

常见扣分点：
- 该加 the 的地方漏了 → *"Government should invest more..."* ❌
- 不该加 the 的地方加了 → *"The technology is important."* ❌（泛指科技）
- 不可数名词加了复数 → *"informations"* ❌ / *"advices"* ❌
- 可数名词单数裸奔 → *"She is teacher."* ❌

</div>

## 学习建议

1. **先学第1章冠词**：这是最核心、最容易出错的部分
2. **再学第2章名词**：可数/不可数直接影响冠词的使用
3. **最后学第3章数量词**：在前两章基础上，学会精确表达数量
4. **大量练习**：冠词没有捷径，只有通过大量练习才能形成语感

::: warning 中国人高频错误
中文 → 英文时，最常犯的错误是**直接翻译**而忽略冠词和名词规则：

- "教育很重要" → ✅ **Education is important.**（不可数名词泛指不加 the，这个中文直译恰好是对的）
- "政府应该投资" → ❌ *Government should invest...* ✅ **The government should invest...**（特指某国政府）
- "她是老师" → ❌ *She is teacher.* ✅ **She is a teacher.**（可数名词单数必须有冠词）
- "我需要一些建议" → ❌ *I need some advices.* ✅ **I need some advice.**（advice 不可数）
:::
