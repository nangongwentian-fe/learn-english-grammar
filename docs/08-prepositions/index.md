# 八、介词与搭配 Prepositions & Collocations

::: tip 为什么介词这么难？
中文说"在三点""在星期一""在一月"，都只用一个"在"字。但英语不一样：**at** 3pm, **on** Monday, **in** January——同样表示"在"，却用了三个不同的介词！更让人头疼的是，介词还会和动词、形容词"捆绑"使用，形成固定搭配（collocation），比如 depend **on**、good **at**。**没有太多规律可循，很多只能靠记忆和大量练习。**

好消息是：雅思考试中最常考的介词和搭配是有限的，掌握本模块的核心内容，足以应对雅思 6.5 分的需求。
:::

## 雅思考试中的介词

- **写作 Task 1**（图表题）：大量使用时间介词描述数据变化 — *"**In** 2020, the figure increased **by** 10%."*
- **写作 Task 2**（议论文）：介词搭配的准确性直接体现词汇分（Lexical Resource）— *"depend **on**"* 而非 ~~*"depend **of**"*~~
- **口语**：介词搭配的流利度和准确度是考官评分的重要依据
- **阅读/听力**：理解介词搭配有助于快速抓住句意

::: warning 介词错误 = 扣分点
在雅思写作和口语评分中，介词搭配错误属于**词汇准确性（Lexical Resource）**扣分项。一个句子如果介词用错，即使整体意思能被理解，也会影响评分。例如：

❌ *"I am interested **of** English."* → ✅ *"I am interested **in** English."*

❌ *"It depends **of** the situation."* → ✅ *"It depends **on** the situation."*
:::

## 学习建议

1. **不要试图"翻译"**：中文的"在"不等于某一个英语介词，必须按英语的思维去记忆
2. **整体记忆搭配**：不要单独记 "depend" 和 "on"，而是把 "depend on" 作为一个整体来记
3. **大量例句输入**：通过读和听来建立语感，比死记硬背规则更有效
4. **错题本**：记录自己犯过的介词错误，定期复习
