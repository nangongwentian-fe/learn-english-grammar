# 三、条件句 Conditionals

::: tip 什么是条件句？
中文里我们说"如果……就……"，英语也一样：**If + 条件, 结果**。但英语的条件句比中文复杂得多——不同的"可能性"要用不同的时态搭配。中文的"如果"不管什么情况都是同一个词，动词不变化；而英语的条件句会根据**事情是真实的还是假设的、是现在的还是过去的**来改变动词形式。
:::

## 本模块包含

| 章节 | 内容 | 核心用途 |
|------|------|---------|
| [零条件句与第一条件句](./01-zero-first) | If + 现在时, 现在时 / will + 动词原形 | 客观事实 / 真实可能的未来 |
| [第二条件句](./02-second) | If + 过去时, would + 动词原形 | 与现在事实相反的假设 |
| [第三条件句与混合条件句](./03-third-mixed) | If + 过去完成时, would have + 过去分词 | 对过去的假设与后悔 / 混合假设 |

## 四种条件句速览

| 条件句类型 | if 从句时态 | 主句 | 含义 | 例句 |
|-----------|-----------|------|------|------|
| 零条件句 | 一般现在时 | 一般现在时 | 客观事实/真理 | If you heat water to 100°C, it **boils**. |
| 第一条件句 | 一般现在时 | will + 动词原形 | 真实可能发生 | If it **rains**, I **will take** an umbrella. |
| 第二条件句 | 一般过去时 | would + 动词原形 | 与现在相反的假设 | If I **had** more time, I **would travel**. |
| 第三条件句 | 过去完成时 | would have + 过去分词 | 与过去相反的假设 | If I **had studied** harder, I **would have passed**. |

## 雅思考试中的条件句

条件句是雅思考试中**提升语法分数的关键句型**，能展示你的假设推理能力：

- **写作 Task 2**（议论文）：用第一条件句讨论可能的结果（*If governments invest more in education, society will benefit.*），用第二条件句提出假设论点（*If everyone recycled, the environment would improve significantly.*）
- **口语 Part 2/3**：用第二条件句表达愿望和假设（*If I had the chance, I would live abroad.*），用第三条件句表达遗憾（*If I had started earlier, I would have achieved more.*）
- **阅读/听力**：理解条件句是正确答题的基础，特别是区分"真实可能"和"假设/虚拟"的语义

::: warning 重要提醒
条件句需要**时态模块**的知识作为基础。如果你还不熟悉一般现在时、一般过去时和过去完成时，建议先学习[时态模块](/01-tenses/)。
:::
