# 零条件句与第一条件句 Zero & First Conditionals

::: tip 用中文理解条件句
中文里"如果下雨，我就带伞"这句话，不管是在说事实还是预测未来，动词"下雨"和"带"都不变。但英语不一样——**说客观事实和说未来可能发生的事，用的时态不同**。零条件句说的是"永远成立的事实"，第一条件句说的是"将来真的可能发生的事"。
:::

---

## 1. 零条件句 Zero Conditional

### 什么是零条件句？

零条件句描述的是**永远成立的客观事实、科学真理、普遍规律**——不是"可能会发生"，而是"一定会发生"。

> 核心公式：**If + 一般现在时, 一般现在时**

想象这些场景：
- 如果你加热水到100°C，水**一定**会沸腾 → 科学事实
- 如果你不浇花，花**一定**会枯死 → 普遍规律
- 如果你按这个按钮，灯**一定**会亮 → 因果关系

### 怎么构成？

#### 肯定句

| if 从句 | 主句 | 例句 |
|---------|------|------|
| If + 主语 + 一般现在时 | 主语 + 一般现在时 | If you **heat** water to 100°C, it **boils**. |
| | | （如果你把水加热到100°C，水会沸腾。）|
| | | If ice **falls** into water, it **floats**. |
| | | （如果冰掉进水里，它会漂浮。）|

#### 否定句

| if 从句 | 主句 | 例句 |
|---------|------|------|
| If + 主语 + don't/doesn't + 动词原形 | 主语 + don't/doesn't + 动词原形 | If you **don't water** plants, they **die**. |
| | | （如果你不浇花，花会死。）|
| | | If the teacher **doesn't come**, the class **doesn't start**. |
| | | （如果老师不来，课不会开始。）|

#### 疑问句

条件句通常不用疑问形式，但可以这样问：

- **What happens** if you mix red and blue? — It **makes** purple.
  （如果你混合红色和蓝色会怎样？——变成紫色。）
- **Does** ice **melt** if you heat it? — Yes, it **does**.
  （如果你加热冰，它会融化吗？——是的。）

### 零条件句 vs 祈使句

零条件句的主句也可以用**祈使句**（命令/建议），表示"在某种情况下应该做什么"：

| 例句 | 中文翻译 |
|------|---------|
| If you feel tired, **take** a break. | 如果你累了，休息一下。 |
| If the alarm rings, **leave** the building immediately. | 如果警报响了，立刻离开大楼。 |
| If you don't understand, **ask** the teacher. | 如果你不明白，问老师。 |

### 零条件句常见例句

| 例句 | 中文翻译 | 场景 |
|------|---------|------|
| If you press this button, the door **opens**. | 你按这个按钮，门就开了。 | 操作说明 |
| If you don't eat, you **get** hungry. | 如果你不吃东西，你会饿。 | 普遍事实 |
| If metal **gets** hot, it **expands**. | 金属受热会膨胀。 | 科学真理 |
| If babies **are** hungry, they **cry**. | 婴儿饿了会哭。 | 普遍规律 |
| If you **touch** fire, you **get** burned. | 如果你碰火，你会被烧伤。 | 因果关系 |

<div class="ielts-tip">

**雅思写作 Task 1**（流程图/说明文）描述流程步骤时可以用零条件句：

- *"If the temperature **reaches** 200°C, the material **changes** its state."*
- *"If the water level **drops** below a certain point, the pump **activates** automatically."*

</div>

### 练习题

<Quiz
  question="If you ___ (heat) ice, it ___ (melt)."
  answer="heat, melts"
  hint="零条件句：if 从句用一般现在时，主句也用一般现在时。ice = it，第三人称单数：melt → melts"
/>

<Quiz
  question="If babies ___ (be) hungry, they ___ (cry)."
  answer="are, cry"
  hint="零条件句两边都是一般现在时。babies = they，be 动词用 are；cry 不用变"
/>

## 2. 第一条件句 First Conditional

### 什么是第一条件句？

第一条件句描述的是**将来真实可能发生的事情**——不是空想，而是说话者认为**有可能发生**的情况。

> 核心公式：**If + 一般现在时, will + 动词原形**

想象这些场景：
- 明天**可能**下雨 → 如果下雨，我就带伞
- 你**可能**会来参加聚会 → 如果你来，我很高兴
- 考试**可能**考得好 → 如果你通过考试，爸爸会给你买手机

### 怎么构成？

#### 肯定句

| if 从句 | 主句 | 例句 |
|---------|------|------|
| If + 主语 + 一般现在时 | 主语 + **will** + 动词原形 | If it **rains** tomorrow, I **will take** an umbrella. |
| | | （如果明天下雨，我会带伞。）|
| | | If you **study** hard, you **will pass** the exam. |
| | | （如果你努力学习，你会通过考试。）|

#### 否定句

| 否定位置 | 构成 | 例句 |
|---------|------|------|
| if 从句否定 | If + 主语 + don't/doesn't | If you **don't hurry**, you **will miss** the bus. |
| | | （如果你不快点，你会错过公交车。）|
| 主句否定 | 主语 + **will not (won't)** | If it **rains**, I **won't go** out. |
| | | （如果下雨，我不出去。）|
| 两边都否定 | | If you **don't study**, you **won't pass**. |
| | | （如果你不学习，你不会通过。）|

#### 疑问句

| 构成 | 例句 |
|------|------|
| **Will** + 主语 + 动词原形 + if...? | **Will** you **come** if I invite you?（如果我邀请你，你会来吗？）|
| What **will** + 主语 + 动词原形 + if...? | What **will** you **do** if it rains?（如果下雨你会做什么？）|

::: warning 中国人高频错误
**if 从句中绝对不能用 will！** 这是中国学生最常犯的错误之一。

中文说"如果明天**会**下雨"，里面有个"会"字，很多人就直接翻译成 will。但在英语条件句中，**if 从句用一般现在时来表示将来**，不用 will。

<div class="example-wrong">❌ If it <strong>will rain</strong> tomorrow, I will take an umbrella.</div>
<div class="example-right">✅ If it <strong>rains</strong> tomorrow, I will take an umbrella.</div>

<div class="example-wrong">❌ If you <strong>will come</strong> to the party, I will be happy.</div>
<div class="example-right">✅ If you <strong>come</strong> to the party, I will be happy.</div>

<div class="example-wrong">❌ If she <strong>will finish</strong> the work, she will go home.</div>
<div class="example-right">✅ If she <strong>finishes</strong> the work, she will go home.</div>

**记忆口诀：if 后面的"将来"用"现在时"表达！**
:::

### if 从句和主句可以换位置

条件句的两部分**可以颠倒顺序**：

| 写法 | 例句 | 注意 |
|------|------|------|
| If 从句在前 | **If** it rains**,** I will stay home. | if 从句在前面时，**中间要加逗号** |
| If 从句在后 | I will stay home **if** it rains. | if 从句在后面时，**不需要逗号** |

两种写法意思完全一样。

### 主句中不一定用 will

除了 will，主句还可以用其他词来表达不同的语气：

| 主句用词 | 含义 | 例句 |
|---------|------|------|
| **will** | 一般将来/确定 | If you ask her, she **will help** you.（她会帮你的。）|
| **can** | 能够 | If you finish early, you **can leave**.（你可以走。）|
| **may/might** | 也许/可能 | If it rains, we **may cancel** the trip.（我们可能取消旅行。）|
| **should** | 应该 | If you feel sick, you **should see** a doctor.（你应该去看医生。）|
| **must** | 必须 | If you want to pass, you **must study**.（你必须学习。）|
| 祈使句 | 命令/建议 | If you see John, **tell** him to call me.（告诉他打电话给我。）|

### 常见例句

| 例句 | 中文翻译 |
|------|---------|
| If I **have** time, I **will visit** you. | 如果我有时间，我会去看你。 |
| If the weather **is** nice, we **will go** to the beach. | 如果天气好，我们去海边。 |
| If you **don't stop** talking, the teacher **will be** angry. | 如果你不停止说话，老师会生气。 |
| If she **gets** the job, she **will move** to Shanghai. | 如果她得到这份工作，她会搬去上海。 |
| If we **leave** now, we **will arrive** on time. | 如果我们现在走，会准时到。 |

<div class="ielts-tip">

**雅思写作 Task 2** 讨论问题的可能后果时大量使用第一条件句：

- *"If governments **invest** more in renewable energy, pollution levels **will decrease** significantly."*
  （如果政府加大对可再生能源的投入，污染水平将会显著下降。）
- *"If students **are given** more freedom in choosing subjects, they **will be** more motivated to learn."*
  （如果给学生更多选课自由，他们会更有学习动力。）
- *"If this trend **continues**, urban areas **will face** severe housing shortages."*
  （如果这一趋势持续下去，城市地区将面临严重的住房短缺。）

**雅思口语 Part 3** 讨论社会问题：

- *"If people **spend** too much time on social media, they **will lose** the ability to communicate face to face."*

</div>

## 3. if vs when — 什么时候用哪个？

很多中国学生分不清 if 和 when，因为中文里"如果"和"当……的时候"有时可以互换。但在英语中，它们有**明确的区别**：

| 词 | 含义 | 确定性 | 例句 |
|---|------|-------|------|
| **if** | 如果（**不确定**会不会发生） | 可能发生，也可能不发生 | **If** it rains, I will stay home.（不知道会不会下雨）|
| **when** | 当……的时候（**确定**会发生） | 一定会发生 | **When** I get home, I will call you.（我一定会到家的）|

### 对比例句

| if（不确定） | when（确定） |
|-------------|-------------|
| **If** I see Tom, I will tell him.（不确定能不能见到 Tom） | **When** I see Tom, I will tell him.（肯定会见到 Tom） |
| **If** it snows, we will build a snowman.（不确定会不会下雪） | **When** it snows, we will build a snowman.（到了冬天肯定会下雪） |
| **If** you come to Beijing, visit me.（不确定你会不会来） | **When** you come to Beijing, visit me.（知道你一定会来） |

::: warning 中国人高频错误
when 后面也**不能用 will**（和 if 一样的规则）！

<div class="example-wrong">❌ When I <strong>will arrive</strong>, I will call you.</div>
<div class="example-right">✅ When I <strong>arrive</strong>, I will call you.</div>

**记忆口诀：if/when 后面的"将来"都用"现在时"表达！**
:::

### 练习题

<Quiz
  question="___ (If / When) I graduate, I will look for a job.（毕业是确定的事）"
  answer="When"
  hint="毕业是确定会发生的事 → 用 When"
/>

<Quiz
  question="___ (If / When) it rains this weekend, we will cancel the picnic.（不确定会不会下雨）"
  answer="If"
  hint="不确定会不会下雨 → 用 If"
/>

## 4. unless = if not

**unless** 的意思就是 **if not**（如果不……/除非）。学会 unless 可以让你的表达更简洁。

### 基本公式

> **Unless + 肯定句 = If + 否定句**

| unless 写法 | if not 写法 | 中文 |
|------------|------------|------|
| **Unless** you study hard, you will fail. | **If** you **don't** study hard, you will fail. | 除非你努力学习，否则你会不及格。 |
| **Unless** it rains, we will go out. | **If** it **doesn't** rain, we will go out. | 除非下雨，否则我们会出去。 |
| I won't go **unless** you come with me. | I won't go **if** you **don't** come with me. | 除非你跟我一起去，否则我不去。 |

::: warning 中国人高频错误
unless 已经包含了"不"的意思，所以 **unless 后面不要再加 not**！

<div class="example-wrong">❌ Unless you <strong>don't</strong> hurry, you will be late.（双重否定，意思反了）</div>
<div class="example-right">✅ Unless you hurry, you will be late.（除非你快点，否则你会迟到。）</div>

<div class="example-wrong">❌ Unless it <strong>doesn't</strong> rain, we will stay inside.</div>
<div class="example-right">✅ Unless it rains, we will stay inside.（除非下雨，否则我们待在屋里。）</div>

**记忆口诀：unless = if not，不要重复否定！**
:::

### 更多例句

| 例句 | 中文翻译 |
|------|---------|
| Unless you **leave** now, you will miss the flight. | 除非你现在走，否则你会错过航班。 |
| I will wait **unless** you **tell** me not to. | 我会等着，除非你叫我别等。 |
| **Unless** the government **takes** action, the problem will get worse. | 除非政府采取行动，否则问题会恶化。 |

<div class="ielts-tip">

**unless** 在雅思写作中是一个**加分词汇**，比 if not 更正式、更地道：

- *"Unless governments **take** immediate action, climate change **will** become irreversible."*
  （除非政府立即采取行动，否则气候变化将不可逆转。）
- *"Young people will struggle to find employment **unless** the education system **is** reformed."*
  （年轻人将难以找到工作，除非教育体制得到改革。）

</div>

### 练习题

<Quiz
  question="将 'If you don't eat breakfast, you will feel tired.' 用 unless 改写。"
  answer="Unless you eat breakfast, you will feel tired."
  hint="unless = if not，去掉 don't：Unless you eat breakfast, you will feel tired."
/>

<Quiz
  question="Unless she ___ (study), she ___ (not / pass) the exam."
  answer="studies, won't pass"
  hint="unless 从句用一般现在时（she → studies），主句：will not (won't) pass"
/>

## 5. 零条件句 vs 第一条件句 — 完整对比

| | 零条件句 Zero Conditional | 第一条件句 First Conditional |
|---|------------------------|--------------------------|
| **公式** | If + 一般现在时, 一般现在时 | If + 一般现在时, will + 动词原形 |
| **含义** | 永远成立的客观事实 | 将来可能发生的事 |
| **确定性** | 100% 确定，永远如此 | 有可能发生，但不确定 |
| **时间** | 不限于某个具体时间 | 指向将来 |
| **例句** | If you heat ice, it **melts**. | If it rains tomorrow, I **will stay** home. |
| **中文** | 如果你加热冰，它会融化。（永远如此） | 如果明天下雨，我会待在家。（不确定会不会下雨） |

### 同一个场景的对比

| 零条件句（事实） | 第一条件句（可能发生） |
|----------------|---------------------|
| If you **eat** too much, you **get** fat.（吃太多就会胖——普遍事实）| If you **eat** too much tonight, you **will feel** sick.（今晚吃太多的话，你会觉得不舒服——具体预测）|
| If you **mix** blue and yellow, you **get** green.（蓝+黄=绿——永远如此）| If you **buy** blue paint, I **will mix** the colors.（你买蓝色颜料的话，我来调色——将来计划）|
| If water **reaches** 0°C, it **freezes**.（水到0°C就结冰——科学事实）| If the temperature **drops** tonight, the roads **will be** icy.（今晚降温的话，路会结冰——具体预测）|

### 练习题（综合判断）

<Quiz
  question="If you ___ (press) this button, the machine ___ (start).（这是操作说明，永远如此）"
  answer="press, starts"
  hint="永远成立的操作说明 → 零条件句：两边都用一般现在时。the machine = it，start → starts"
/>

<Quiz
  question="If she ___ (call) me tonight, I ___ (tell) her the news.（不确定她会不会打电话）"
  answer="calls, will tell"
  hint="不确定会不会发生 → 第一条件句：if 从句用一般现在时（calls），主句用 will tell"
/>

## 本章总结

### 关键规则清单

1. **零条件句**：If + 一般现在时, 一般现在时 → 客观事实、科学真理
2. **第一条件句**：If + 一般现在时, will + 动词原形 → 真实可能的将来
3. **if 从句永远不用 will**（最重要的规则！）
4. **when 用于确定会发生的事，if 用于不确定的事**
5. **unless = if not**，unless 后面不要再加 not

### 核心对比表

| | 零条件句 | 第一条件句 |
|---|---------|----------|
| if 从句 | 一般现在时 | 一般现在时 |
| 主句 | 一般现在时 | will + 动词原形 |
| 含义 | 永远的事实 | 可能的将来 |
| 时间标志 | 无特定时间 | tomorrow, next week, tonight... |

::: warning 本章最重要的一条规则
**if 从句中不能用 will！** 这是中国学生最常犯的错误，也是雅思考试中的常见扣分点。

<div class="example-wrong">❌ If it <strong>will</strong> rain, I will stay home.</div>
<div class="example-right">✅ If it <strong>rains</strong>, I will stay home.</div>

看到 if，就把"将来"用"现在时"表达。反复练习，形成条件反射！
:::
