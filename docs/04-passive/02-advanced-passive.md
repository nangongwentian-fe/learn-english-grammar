# 进阶被动语态 Advanced Passive

::: tip 掌握进阶被动，写作口语都加分
学完了基本被动语态（be + 过去分词），你已经能应付大部分场景了。但雅思考试中还有一些**高级被动结构**，掌握它们能让你的表达更丰富、更地道。这一节我们学习：情态动词+被动、get 被动、"据说"句型、have something done、以及**什么时候该用被动语态**。
:::

---

## 1. 情态动词 + 被动语态

### 核心公式

> **情态动词 + be + 过去分词**

情态动词（can, could, may, might, must, should, will, would 等）后面永远接**动词原形**，所以被动中 be 保持原形 **be**（不变成 is/are/was 等）。

### 各情态动词的被动形式

| 情态动词 | 被动形式 | 例句 | 中文 |
|---------|---------|------|------|
| **can** | can be + done | This problem **can be solved** easily. | 这个问题能被轻松解决。 |
| **could** | could be + done | The meeting **could be postponed**. | 会议可能被推迟。 |
| **may** | may be + done | Students **may be asked** to leave. | 学生可能被要求离开。 |
| **might** | might be + done | The flight **might be delayed**. | 航班可能被延误。 |
| **must** | must be + done | Homework **must be submitted** by Friday. | 作业必须在周五之前提交。 |
| **should** | should be + done | Children **should be taught** good manners. | 孩子应该被教导良好的礼仪。 |
| **will** | will be + done | The project **will be completed** next month. | 项目将在下月完成。 |
| **would** | would be + done | He **would be given** a second chance. | 他会被给予第二次机会。 |
| **have to** | have to be + done | This form **has to be filled** in correctly. | 这个表格必须被正确填写。 |
| **ought to** | ought to be + done | More money **ought to be spent** on education. | 更多的钱应该花在教育上。 |
| **need to** | need to be + done | The car **needs to be washed**. | 这辆车需要被洗。 |

### 否定句

在情态动词后加 not：

- Phones **cannot (can't) be used** during the exam.（考试期间不能使用手机。）
- The data **should not (shouldn't) be shared** publicly.（数据不应该被公开分享。）
- This issue **must not (mustn't) be ignored**.（这个问题不能被忽视。）

### 疑问句

情态动词提到主语前：

- **Can** this problem **be fixed**?（这个问题能被修好吗？）
- **Should** the meeting **be rescheduled**?（会议应该被改期吗？）
- **Must** all documents **be submitted** today?（所有文件必须今天提交吗？）

::: warning 中国人高频错误
**情态动词后的 be 不能变形！** 永远是原形 **be**：

<div class="example-wrong">❌ This must is done immediately.</div>
<div class="example-right">✅ This must <strong>be done</strong> immediately.（must 后面只能接原形 be）</div>

<div class="example-wrong">❌ The work should been finished by now.</div>
<div class="example-right">✅ The work should <strong>be</strong> finished by now.（should 后面接 be，不是 been）</div>

<div class="example-wrong">❌ The rules can not be break.</div>
<div class="example-right">✅ The rules cannot be <strong>broken</strong>.（break 的过去分词是 broken，不是 break）</div>
:::

<div class="ielts-tip">

**情态动词 + 被动是雅思写作 Task 2 的高分句型：**

提建议时：
- *"Stricter regulations **should be enforced** to reduce pollution."*（应该实施更严格的法规来减少污染。）
- *"More funding **should be allocated** to medical research."*（更多资金应该被分配给医学研究。）

表达可能性时：
- *"This trend **can be attributed** to several factors."*（这个趋势可以归因于几个因素。）
- *"It **could be argued** that technology has more advantages than disadvantages."*（可以说技术的优势大于劣势。）

表达必要性时：
- *"Urgent action **must be taken** to address climate change."*（必须采取紧急行动应对气候变化。）
- *"The root causes **need to be identified** before solutions can be proposed."*（在提出解决方案之前，需要找出根本原因。）

</div>

### 练习题

<Quiz
  question="The homework ___ (must / submit) by Friday."
  answer="must be submitted"
  hint="情态动词 + be + 过去分词：must + be + submitted"
/>

<Quiz
  question="Mobile phones ___ (should / not / use) in the classroom."
  answer="should not be used"
  hint="情态动词 + not + be + 过去分词：should not be used"
/>

<Quiz
  question="This problem ___ (can / solve) in many different ways."
  answer="can be solved"
  hint="情态动词 + be + 过去分词：can be solved"
/>

## 2. get + 过去分词（口语被动）

### 什么是 get 被动？

除了 **be + 过去分词**，英语口语中还经常用 **get + 过去分词** 来表示被动，语气更口语化、更生动。

| be 被动（正式） | get 被动（口语） | 中文 |
|----------------|----------------|------|
| He **was injured** in the accident. | He **got injured** in the accident. | 他在事故中受伤了。 |
| She **was promoted** last month. | She **got promoted** last month. | 她上个月升职了。 |
| My phone **was stolen**. | My phone **got stolen**. | 我的手机被偷了。 |
| We **were invited** to the party. | We **got invited** to the party. | 我们被邀请去参加派对。 |

### get 被动 vs be 被动的区别

| | be 被动 | get 被动 |
|---|---------|---------|
| **正式程度** | 正式、书面 | 口语、非正式 |
| **语气** | 中性、客观 | 更生动、更强调"变化过程" |
| **雅思写作** | ✅ 适合 | ❌ 太口语，避免使用 |
| **雅思口语** | ✅ 适合 | ✅ 适合，显得自然 |

### get 被动常见表达

以下是日常口语中非常高频的 get 被动表达：

| 表达 | 中文 | 例句 |
|------|------|------|
| get married | 结婚 | They **got married** in June.（他们六月结的婚。） |
| get divorced | 离婚 | She **got divorced** last year.（她去年离婚了。） |
| get lost | 迷路 | We **got lost** in the forest.（我们在森林里迷路了。） |
| get hurt / injured | 受伤 | He **got hurt** playing football.（他踢足球受伤了。） |
| get fired | 被解雇 | Tom **got fired** from his job.（汤姆被炒鱿鱼了。） |
| get paid | 被支付/拿到工资 | I **get paid** every Friday.（我每周五发工资。） |
| get stuck | 被卡住 | The car **got stuck** in the mud.（车陷在泥里了。） |
| get caught | 被抓到 | He **got caught** cheating in the exam.（他考试作弊被抓了。） |
| get invited | 被邀请 | She **got invited** to the conference.（她被邀请参加会议。） |
| get arrested | 被逮捕 | The thief **got arrested** this morning.（小偷今早被逮捕了。） |

::: warning 中国人高频错误
**get 被动在雅思写作中不要用！** 写作中请用 be 被动。

<div class="example-wrong">❌（写作中）Many workers got fired during the recession.</div>
<div class="example-right">✅（写作中）Many workers <strong>were dismissed</strong> during the recession.</div>

但**口语中用 get 被动很加分**，比 be 被动更自然：

<div class="example-right">✅（口语中）I almost <strong>got hit</strong> by a car yesterday!（我昨天差点被车撞了！）</div>
:::

<div class="ielts-tip">

**雅思口语**中使用 get 被动可以让回答更自然：

Part 1:
- *"I usually **get paid** at the end of each month."*（我通常月底发工资。）
- *"I **got interested** in photography when I was in high school."*（我高中时对摄影产生了兴趣。）

Part 2 讲故事：
- *"We **got lost** during the trip, which was actually quite fun."*（旅途中我们迷路了，其实挺好玩的。）
- *"I **got selected** to represent my school in the competition."*（我被选中代表学校参加比赛。）

</div>

  answer="He got injured in the match."
  hint="把 was 换成 got：He got injured in the match."

## 3. It is said that... / He is said to...（据说...）

### 转述结构（Reporting Passive）

英语中有一类特殊的被动结构，用来转述别人的观点、研究结论或普遍看法，而**不指明具体是谁说的**。这在学术写作中**极其常见**。

### 结构一：It + be + 过去分词 + that 从句

| 动词 | 被动结构 | 中文 | 例句 |
|------|---------|------|------|
| say | It **is said** that... | 据说... | It **is said** that he is very rich.（据说他很有钱。） |
| believe | It **is believed** that... | 人们认为... | It **is believed** that exercise improves health.（人们认为运动改善健康。） |
| think | It **is thought** that... | 人们觉得... | It **is thought** that the plan will succeed.（人们觉得这个计划会成功。） |
| know | It **is known** that... | 众所周知... | It **is known** that smoking causes cancer.（众所周知吸烟致癌。） |
| report | It **is reported** that... | 据报道... | It **is reported** that 100 people were injured.（据报道100人受伤。） |
| estimate | It **is estimated** that... | 据估计... | It **is estimated** that 1 billion people lack clean water.（据估计10亿人缺乏干净的水。） |
| expect | It **is expected** that... | 预计... | It **is expected** that prices will rise.（预计物价将上涨。） |
| suggest | It **is suggested** that... | 有人建议... | It **is suggested** that students read more.（有人建议学生多读书。） |
| argue | It **is argued** that... | 有人认为... | It **is argued** that technology does more harm than good.（有人认为科技弊大于利。） |
| claim | It **is claimed** that... | 有人声称... | It **is claimed** that the drug has no side effects.（有人声称这种药没有副作用。） |
| consider | It **is considered** that... | 人们认为... | It **is considered** that education is essential.（人们认为教育是必不可少的。） |
| assume | It **is assumed** that... | 人们假设... | It **is assumed** that everyone has equal access.（人们假设每个人都有平等的机会。） |

也可以用**过去时**：

- It **was believed** that the Earth was flat.（人们曾经认为地球是平的。）
- It **was reported** that the company had gone bankrupt.（据报道该公司已经破产。）

### 结构二：主语 + be + 过去分词 + to + 动词原形

上面的 "It is said that..." 可以转换成另一种更简洁的形式，把 that 从句的主语提到句首：

| It... that... 结构 | 主语 + be + 过去分词 + to... 结构 |
|-------------------|--------------------------------|
| It is said **that he is** a genius. | **He is said to be** a genius. |
| It is believed **that she has** left the country. | **She is believed to have left** the country. |
| It is reported **that they are** investigating the case. | **They are reported to be investigating** the case. |
| It was thought **that he had** stolen the money. | **He was thought to have stolen** the money. |

转换规则：
1. that 从句的主语 → 变成整个句子的主语
2. 保留 be + 过去分词
3. that 从句的谓语 → 变成 **to + 动词原形**（同一时间）或 **to have + 过去分词**（更早的时间）

::: warning 中国人高频错误
**to 后面的动词形式要注意：**

- 如果 that 从句和主句**时间一致**，用 **to + 动词原形 / to be doing**
- 如果 that 从句的时间**更早**，用 **to have + 过去分词**

<div class="example-wrong">❌ He is said to left the country.（left 是过去式，不是原形）</div>
<div class="example-right">✅ He is said <strong>to have left</strong> the country.（离开发生在"说"之前 → to have left）</div>

<div class="example-wrong">❌ She is believed that she is innocent.</div>
<div class="example-right">✅ She is believed <strong>to be</strong> innocent.（不能同时用 She is believed 又加 that she...）</div>
:::

<div class="ielts-tip">

**这类转述被动结构是雅思写作 Task 2 的万能开头句：**

引出普遍观点：
- *"It **is widely believed** that a university education is essential for career success."*（人们普遍认为大学教育对职业成功至关重要。）
- *"It **is often argued** that governments should invest more in public healthcare."*（人们经常争论政府应该在公共医疗上投入更多。）

引用数据/研究：
- *"It **is estimated** that by 2050, the world population will reach 10 billion."*（据估计到2050年，世界人口将达到100亿。）
- *"It **has been suggested** that remote working improves productivity."*（有人提出远程工作能提高生产力。）

这些句型比 "Many people think that..." 更高级、更学术，能帮助你在 **Grammatical Range** 这一项拿到更高的分数。

</div>

### 练习题

<Quiz

  answer="It is said that he is very intelligent."
  hint="It is said that + 完整句子：It is said that he is very intelligent."
/>

<Quiz
  question="把 'It is believed that she has moved to London.' 改写成 She... 开头的形式。"
  answer="She is believed to have moved to London."
  hint="搬到伦敦发生在'被认为'之前 → to have moved"
/>

<Quiz
  question="据报道100人在地震中受伤。用 It is reported that... 翻译。"
  answer="It is reported that 100 people were injured in the earthquake."
  hint="It is reported that + 完整句子（受伤发生在过去 → were injured）"
/>

## 4. have something done（让别人做某事）

### 什么是 have something done？

这个结构表示"**安排/让别人为你做某事**"——你不是亲自做的，而是请专业的人来做的。

中文里我们说：
- 我**剪了**头发。→ 其实是别人帮你剪的，不是你自己拿剪刀剪的！
- 我**修了**车。→ 其实是修车师傅修的。

英语中也有这个区别：

| 自己做 | 让别人做 |
|--------|---------|
| I **cut** my hair.（我自己剪了头发——很少见） | I **had** my hair **cut**.（我让人帮我剪了头发——去了理发店） |
| She **repaired** her car.（她自己修了车） | She **had** her car **repaired**.（她让人帮她修了车——去了修车店） |

### 构成

> **have + 宾语 + 过去分词**

| 时态 | 构成 | 例句 | 中文 |
|------|------|------|------|
| 一般现在时 | have/has + sth + done | I **have** my car **serviced** every year. | 我每年让人保养一次车。 |
| 一般过去时 | had + sth + done | She **had** her hair **cut** yesterday. | 她昨天去剪了头发。 |
| 一般将来时 | will have + sth + done | I **will have** my eyes **tested** next week. | 我下周要去检查视力。 |
| 现在完成时 | have/has had + sth + done | He **has had** his house **painted**. | 他已经让人把房子粉刷了。 |
| 现在进行时 | am/is/are having + sth + done | We **are having** the kitchen **renovated**. | 我们正在让人翻修厨房。 |
| 情态动词 | should/must/need to have + sth + done | You **should have** your teeth **checked** regularly. | 你应该定期去检查牙齿。 |

### 常见使用场景

| 场景 | 例句 | 中文 |
|------|------|------|
| 理发 | I need to **have** my hair **cut**. | 我需要去剪头发。 |
| 修车 | She **had** her car **repaired** last week. | 她上周去修了车。 |
| 洗衣服 | I **have** my suits **dry-cleaned**. | 我把西装送去干洗。 |
| 检查视力 | You should **have** your eyes **tested**. | 你应该去检查视力。 |
| 拍照 | We **had** our photo **taken** at the studio. | 我们在照相馆拍了照。 |
| 装修 | They are **having** the house **painted**. | 他们正在让人粉刷房子。 |
| 送快递 | I'll **have** the package **delivered** to your office. | 我会让人把包裹送到你办公室。 |
| 看牙 | I need to **have** a tooth **pulled out**. | 我需要去拔一颗牙。 |

### get something done（口语替代形式）

在口语中，**get something done** 和 **have something done** 意思完全一样：

| have sth done（稍正式） | get sth done（口语） |
|------------------------|---------------------|
| I **had** my car **repaired**. | I **got** my car **repaired**. |
| She **had** her nails **done**. | She **got** her nails **done**. |
| You should **have** it **fixed**. | You should **get** it **fixed**. |

### have something done 的另一个意思：遭遇不幸

**have something done** 还可以表示"**某人遭遇了不好的事**"（被动的、不是自己安排的）：

- I **had** my wallet **stolen** on the bus.（我的钱包在公交车上被偷了。）——不是我安排的！
- She **had** her house **broken into** last night.（她家昨晚被闯入了。）
- He **had** his leg **broken** in the accident.（他在事故中腿断了。）

> 怎么区分两种意思？看**语境**：
> - "I had my hair cut." → 去理发店（安排别人做 → 正常行为）
> - "I had my bag stolen." → 包被偷了（遭遇不幸 → 不是你愿意的）

::: warning 中国人高频错误
**过去分词不能变成过去式或-ing形式！**

<div class="example-wrong">❌ I had my hair cutting yesterday.</div>
<div class="example-right">✅ I had my hair <strong>cut</strong> yesterday.（cut 的过去分词就是 cut）</div>

<div class="example-wrong">❌ She had her car to repair.</div>
<div class="example-right">✅ She had her car <strong>repaired</strong>.（宾语后接过去分词，不是 to + 动词）</div>

<div class="example-wrong">❌ I have my room clean every week.</div>
<div class="example-right">✅ I have my room <strong>cleaned</strong> every week.（clean 的过去分词是 cleaned）</div>

**另外注意：** 不要把 "have sth done" 和 "have done sth（现在完成时）" 搞混：

- I **have repaired** my car.（现在完成时 → 我自己修了车）
- I **have** my car **repaired**.（have sth done → 我让别人修了车）
:::

<div class="ielts-tip">

**雅思口语**中 have sth done 非常实用：

Part 1（日常话题）:
- *"I usually **have** my hair **cut** once a month."*（我通常一个月剪一次头发。）
- *"I **had** my phone screen **replaced** recently because I dropped it."*（我最近换了手机屏幕，因为我摔了手机。）

Part 2（描述经历）:
- *"I decided to **have** the old house **renovated** rather than buy a new one."*（我决定让人翻修旧房子而不是买新的。）

Part 3（深度讨论）:
- *"Many people prefer to **have** their clothes **made** by tailors rather than buying off the rack."*（很多人喜欢让裁缝做衣服而不是买成衣。）

</div>

## 5. 主动与被动的选择

这是**雅思写作最关键的知识点之一**：什么时候该用主动，什么时候该用被动？

### 用被动语态的场景

| 场景 | 为什么用被动 | 例句 |
|------|------------|------|
| **执行者不重要/不知道** | 关注的是动作或结果 | My bike **was stolen**.（不知道谁偷的） |
| **执行者是"人们"泛指** | 避免写 "People..." 太笼统 | English **is spoken** worldwide. |
| **学术/正式写作** | 让语气更客观 | The experiment **was conducted** over 6 months. |
| **描述流程/步骤** | 关注的是每一步做了什么 | The materials **are heated** and then **are shaped**. |
| **描述数据变化（Task 1）** | 关注的是数据本身 | A decrease **was observed** between 2010 and 2015. |
| **想强调承受者** | 承受者是谈话的重点 | The Mona Lisa **was painted** by Da Vinci. |

### 用主动语态的场景

| 场景 | 为什么用主动 | 例句 |
|------|------------|------|
| **执行者很重要** | 明确说是谁做的 | **The government** should invest more in education. |
| **句子更简洁** | 主动比被动更短 | 主动: We found the answer. / 被动: The answer was found by us.（多余） |
| **日常口语** | 更自然 | I ate lunch at noon.（而不是 Lunch was eaten by me...） |
| **表达观点** | 更直接有力 | **I strongly believe** that...（而不是 It is strongly believed by me...） |

### 雅思写作中的主动 vs 被动

::: warning 重要原则
**不要过度使用被动！** 好的写作是主动和被动**混合使用**的。

全部用被动 → 文章读起来很生硬、很别扭
全部用主动 → 语法不够多样化，分数会受限

**理想比例：** 大约 70% 主动 + 30% 被动
:::

### 实战对比：同一个观点的不同写法

**话题：政府是否应该投入更多钱到教育上？**

#### 全部主动（不够学术）

> *Many people think governments should spend more money on education. Experts say this can improve the economy. The government can create more jobs by investing in schools.*

#### 全部被动（太生硬）

> *It is thought by many people that more money should be spent on education by governments. It is said by experts that the economy can be improved. More jobs can be created by the government by investing in schools.*

#### 主被动混合（最佳）

> *It **is widely believed** that governments **should invest** more in education. Research **has shown** that increased spending on schools **can lead** to economic growth. Furthermore, better education **is often associated** with lower unemployment rates.*

<div class="ielts-tip">

**雅思写作主被动混合的黄金搭配：**

**开头引出话题（用被动更客观）：**
- *"It **is often argued** that..."*
- *"It **has been widely debated** whether..."*
- *"Much attention **has been paid** to the issue of..."*

**表达自己观点（用主动更直接）：**
- *"I **strongly believe** that..."*
- *"This essay **will argue** that..."*
- *"The evidence **suggests** that..."*

**描述数据（Task 1 主被动都可以）：**
- 主动: *"The graph **shows** a sharp increase."*
- 被动: *"A sharp increase **is shown** in the graph."*
- 主动: *"The number of students **rose** dramatically."*
- 被动: *"A dramatic rise **was recorded** in student numbers."*

**提出建议（用被动更委婉）：**
- *"More resources **should be allocated** to..."*
- *"Stricter measures **need to be implemented** to..."*
- *"This issue **must be addressed** urgently."*

**总结（主动被动混合）：**
- *"In conclusion, while technology **has brought** many benefits, potential risks **should not be overlooked**."*

</div>

## 6. 特殊被动结构

### 6.1 双宾语动词的被动

有些动词后面可以接**两个宾语**（间接宾语 = 人，直接宾语 = 物）。这些动词变被动时，**两个宾语都可以做主语**：

| 主动 | 被动1（人做主语） | 被动2（物做主语） |
|------|-----------------|-----------------|
| They gave **me** **a book**. | **I** was given a book. | **A book** was given to me. |
| She told **us** **the news**. | **We** were told the news. | **The news** was told to us. |
| He offered **her** **a job**. | **She** was offered a job. | **A job** was offered to her. |

> **在英语中，"人做主语"的被动更常用、更自然。**

常见的双宾语动词：give, tell, show, teach, offer, send, pay, ask, lend

<div class="example-wrong">❌ A prize was given for the winner.</div>
<div class="example-right">✅ A prize was given <strong>to</strong> the winner. / The winner was given a prize.</div>

### 6.2 动词短语的被动

有些动词和介词/副词搭配构成"动词短语"，变被动时**短语不能拆开**：

| 动词短语 | 主动 | 被动 |
|---------|------|------|
| look after（照顾） | Nurses **look after** patients. | Patients **are looked after** by nurses. |
| bring up（抚养） | Her grandparents **brought her up**. | She **was brought up** by her grandparents. |
| put off（推迟） | They **put off** the meeting. | The meeting **was put off**. |
| carry out（执行） | Scientists **carried out** the experiment. | The experiment **was carried out** by scientists. |
| turn down（拒绝） | They **turned down** my application. | My application **was turned down**. |
| look into（调查） | The police **are looking into** the case. | The case **is being looked into** by the police. |
| deal with（处理） | We must **deal with** this problem. | This problem must **be dealt with**. |
| take care of（照顾） | She **takes care of** her mother. | Her mother **is taken care of** by her. |

::: warning 中国人高频错误
动词短语变被动时，**介词不能丢！**

<div class="example-wrong">❌ The children were looked by their grandmother.</div>
<div class="example-right">✅ The children were looked <strong>after</strong> by their grandmother.（look after 不能拆）</div>

<div class="example-wrong">❌ This problem must be dealt.</div>
<div class="example-right">✅ This problem must be dealt <strong>with</strong>.（deal with 不能拆）</div>
:::

## 7. 各种被动结构总结表

| 结构 | 公式 | 例句 | 适用场景 |
|------|------|------|---------|
| 基本被动 | be + done | English **is spoken** here. | 通用 |
| 情态动词被动 | 情态动词 + be + done | It **must be done** now. | 表达必须/可能/应该 |
| 进行时被动 | be + being + done | It **is being repaired**. | 正在被做 |
| 完成时被动 | have/has + been + done | It **has been finished**. | 已经被做 |
| get 被动 | get + done | He **got injured**. | 口语 |
| It is said that... | It + be + done + that... | It **is said** that... | 学术写作转述 |
| 主语 + is said to... | 主语 + be + done + to... | He **is said to be**... | 学术写作转述 |
| have sth done | have + 宾语 + done | I **had** my hair **cut**. | 安排别人做/遭遇不幸 |
| 双宾语被动 | 人/物 + be + done | She **was given** a book. | 双宾语动词 |
| 动词短语被动 | be + 动词短语 | It **was looked into**. | 动词短语 |
