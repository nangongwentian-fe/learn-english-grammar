# 九、句子结构 Sentence Structure

::: tip 为什么句子结构如此重要？
中文是"意合"语言——只要意思到了，语序和连接词可以比较灵活。英语是"形合"语言——**句子必须有严格的结构**，主语、谓语缺一不可，句子之间必须用连接词或标点正确连接。如果你直接按照中文的思维方式写英语句子，很容易写出"碎片句"或"粘连句"，这在雅思写作中会被严重扣分。
:::

## 本模块概览

句子结构是英语语法的"骨架"。无论你学了多少时态、从句、被动语态，最终都要落实到**句子**上。掌握正确的句子结构，是写出正确、地道英语的基础。

## 雅思写作与句子结构

<div class="ielts-tip">

**雅思写作评分标准（Grammatical Range and Accuracy）明确要求：**

- **Band 5**：只能使用简单句，偶尔尝试复杂句但频繁出错
- **Band 6**：能使用简单句和复杂句的混合，但有一些错误
- **Band 7**：能使用多种复杂结构，大部分句子准确无误
- **Band 8+**：灵活使用各种复杂结构，错误极少

**结论：想要 6.5 分，你必须能熟练写出正确的并列句和复合句。** 仅靠简单句堆砌，最多只能拿到 5.5 分。

</div>

## 学习建议

1. **先打好基础**：从五大基本句型开始，确保每个简单句都能写对
2. **再学组合**：掌握并列句和复合句的写法，学会用连接词把简单句组合成复杂句
3. **最后排错**：对照常见错误清单，检查自己的写作中是否有同类问题
4. **大量练习**：每学完一节，完成所有 Quiz 练习题，巩固记忆
