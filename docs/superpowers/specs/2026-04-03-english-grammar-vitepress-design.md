# 中文友好雅思英语语法电子书 — VitePress 设计文档

**日期：** 2026-04-03
**目标用户：** 备考雅思（目标6.5分）的中国学习者
**核心目标：** 将纸质语法书电子化，随时可查、可搜索、可练习

---

## 1. 项目概述

基于 VitePress 构建一本中英双语英语语法电子书，内容结构参考 Murphy's *English Grammar in Use*（145单元）和 Collins *Grammar for IELTS*（25单元），融合为9大模块。每个语法点包含规则讲解、例句对比、雅思应用场景和交互式练习题。

---

## 2. 章节结构

| 模块 | 目录名 | 主要内容 |
|------|--------|---------|
| 一、时态 | `01-tenses` | 现在时 / 过去时 / 将来时（各细分时态） |
| 二、情态动词 | `02-modals` | can/could/may/might/must/should/would |
| 三、条件句 | `03-conditionals` | 零/一/二/三条件句、混合条件句、倒装条件句 |
| 四、被动语态 | `04-passive` | 各时态被动形式、主动vs被动选择、学术写作用法 |
| 五、非谓语动词 | `05-non-finite` | 不定式 / 动名词 / 分词短语 |
| 六、从句 | `06-clauses` | 定语从句 / 名词性从句 / 状语从句 |
| 七、冠词与名词 | `07-articles` | a/an/the / 可数不可数名词 / 零冠词 |
| 八、介词与搭配 | `08-prepositions` | 时间/地点介词 / 动词形容词 + 介词搭配 |
| 九、句子结构 | `09-sentence-structure` | 简单句/复合句 / 倒装句 / 强调句 |

### 每个单元页面结构（固定模板）

1. **规则讲解** — 英文规则 + 中文解释（对照排版）
2. **例句对比** — ❌ 错误示例 vs ✅ 正确示例，附中文说明
3. **雅思应用** — 该语法点在雅思写作/口语中的具体用法和评分关联
4. **练习题** — `<Quiz>` 组件，点击才展开答案和提示

### Quiz 组件接口

```vue
<Quiz
  question="She ___ (go) to school every day."
  answer="goes"
  hint="一般现在时，第三人称单数需要加 -s"
/>
```

- `question`：题目，用 `___` 表示填空位置
- `answer`：正确答案
- `hint`：中文提示（解释为什么）

点击"查看答案"后展示答案和提示，支持再次隐藏。

## 4. 部署方案

- **平台：** Vercel，绑定 GitHub 仓库，push 自动触发构建
- **构建命令：** `npm run docs:build`
- **输出目录：** `.vitepress/dist`
- **访问方式：** Vercel 提供的免费域名（可后续绑定自定义域名）

---

## 5. 设计原则

- **内容优先**：框架选择服务于内容创作，不引入不必要的复杂度
- **中文友好**：规则讲解、错误提示、练习提示全部提供中文
- **雅思导向**：每章节的"雅思应用"模块直接对应考试得分点
- **YAGNI**：不做进度追踪、用户系统、评论等功能，专注语法内容
