# 五、非谓语动词 Non-finite Verbs

::: tip 什么是"非谓语动词"？
中文里没有这个概念，所以先用最简单的方式理解：

英语句子里**只能有一个谓语动词**（就是句子的"主要动作"）。但是如果你想在一个句子里放**第二个动作**，这个动作就不能再当谓语了——它要变成**非谓语动词**。

非谓语动词有三种形式：
- **不定式**（to do）— I want **to learn** English.
- **动名词**（doing）— I enjoy **learning** English.
- **分词**（doing / done）— The **running** man is my brother.

简单说：**非谓语动词 = 动词换了个"身份"，去充当名词、形容词或副词。**
:::

## 为什么非谓语动词对雅思很重要？

非谓语动词是雅思 **6.5 分以上**的必备语法工具：

- **写作 Task 2**：用不定式表目的（*in order to improve...*）、用动名词作主语（*Learning a language requires...*）能让句子更加学术化
- **写作 Task 1**：用分词短语描述图表（*The percentage of students studying abroad increased...*）
- **口语 Part 2/3**：灵活使用非谓语动词能显著提升"语法多样性"评分（Grammatical Range）
- **阅读**：长难句中大量使用非谓语结构，不理解就读不懂

<div class="ielts-tip">

雅思评分标准中，**Grammatical Range and Accuracy（语法广度与准确性）** 占写作和口语总分的 25%。非谓语动词是从 5.5 分突破到 6.5 分的**关键语法点**——因为它能让你的句子更紧凑、更学术、更地道。

</div>

## 三种非谓语动词速览对比

| | 不定式 to do | 动名词 doing | 分词 doing / done |
|---|---|---|---|
| **长什么样** | to + 动词原形 | 动词 + ing | 动词 + ing（现在分词）/ 动词 + ed（过去分词） |
| **相当于什么词** | 名词 / 形容词 / 副词 | 名词 | 形容词 / 副词 |
| **典型例句** | I want **to study**. | **Studying** is important. | The **broken** window... |
| **核心功能** | 表目的、表意愿 | 表一般性动作 | 表描述、表伴随 |

## 学习建议

1. **先学不定式**——最常用，规则最明确
2. **再学动名词**——重点记忆"哪些动词后面跟 doing"
3. **最后学分词**——难度最高，但雅思写作加分利器
4. **反复对比**——to do 和 doing 的区别是考试高频考点
